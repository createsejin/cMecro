Kill Chain GTA5 ver1.0.0

  본 프로그램은 호스트의 서버 프로그램에 멤버들이 클라이언트 프로그램으로 접속하여, 유사시 간단한 핫키 입력을 통해
서로간의 신호를 주고받아 서버 프로그램에 접속된 멤버들의 GTA 프로세스를 거의 동시에 강제로 종료하게끔 하는
프로그램입니다. 그 이외의 목적으로 제작하지 않았으며, 절대로 핵이나 해킹 프로그램이 아닙니다.
  그럼에도 본 프로그램을 신뢰할 수 없다면 실행하지 마십시오.

  본 프로그램은 Windows10 64bit 환경에서 제작되었습니다. 다른 OS 환경에서의 사용은 오류를 불러일으킬 수 있으므로
신중히 사용하시고, 에러 발생시 에러 로그를 아래의 이메일로 보내주세요.


------------------------------------------------ 프로그램 사용 설명 ------------------------------------------------------
https://gall.dcinside.com/mgallery/board/view?id=rsg&no=564929 참조

-> 호스트
  호스트가 서버 프로그램을 실행해야 멤버들이 클라이언트 프로그램으로 호스트의 서버 프로그램에 접속 할 수 있습니다.

  호스트는 자신의 공인 IP를 멤버들에게 공개하여 멤버들이 자신의 서버 프로그램에 접속할 수 있도록 하십시오.
공인 IP는 화세 프로그램이나 네이버에서 'ip 주소 확인'을 검색하면 알 수 있습니다.
만약 호스트의 네트워크가 공유기에 연결된 상태라면 외부에서 호스트의 공인 IP로 서버 프로그램에 접근할 수 없으므로,
공유기의 포트포워딩 설정을 통해 외부 멤버들이 서버 프로그램에 접근할 수 있도록 설정하십시오. 포트포워딩은
설치된 공유기에 따라 상이할 수 있으므로 적절히 검색하여 설정해주시기 바랍니다. 포트포워딩 포트번호는 반드시
'12345'로 설정해 주십시오.

  서버 프로그램 최초 실행시 윈도우 방화벽이 네트워크 연결을 허용할 것인지 묻는 대화창이 나타납니다.
'액세스 허용'을 눌러주십시오.

  서버 프로그램은 자동으로 자신의 공인 IP 혹은 공유기의 사설 IP를 내부적으로 입력하므로 서버 프로그램 실행시에는
자신의 IP를 입력할 필요는 없습니다.

  호스트는 반드시 서버 프로그램과 함께 클라이언트 프로그램도 같이 실행하여 자신의 서버에 접속하셔야 멤버들간의
신호 통신이 가능해집니다. 호스트는 반드시 클라이언트 프로그램도 같이 실행하여 자신의 서버에 접속해주십시오.

  서버 프로그램은 server 폴더의 server.exe를 실행하시고, 클라이언트 프로그램은 client 폴더의 client.exe를
실행하십시오. 클라이언트 프로그램은 실행 후 반드시 호스트 자신의 공인 IP를 입력하여 자신의 서버에 접속해야 합니다.
호스트와 멤버들은 프로그램 초기에 입력된 nickname을 통해 서로의 접속 현황을 체크할 수 있습니다. 반드시 실행전
멤버들의 접속 유무를 체크해주십시오. 접속 여부는 멤버들이 접속하거나 접속을 종료할때 자동으로 프로그램에 출력됩니다.


-> 멤버 + 호스트
  멤버와 호스트는 client 폴더의 client.exe를 실행하여 호스트의 공인 IP를 통해 호스트의 서버 프로그램에 접속하여 멤버들의
접속 여부를 체크하기 바랍니다. 멤버들의 접속 여부는 멤버들이 접속하거나 접속을 종료할때 자동으로 프로그램에 출력됩니다.

  호스트의 서버 프로그램에 정상적으로 접근하였다면 F2키를 눌러 각 멤버의 신호 송수신 상태를 체크할 수 있습니다.

  프로그램을 사용하기 전에 반드시 멤버들과 상의하여 테스트로 GTA가 정상적으로 동시에 종료되는지를 확인한 뒤
사용하십시오.

  GTA와 함께 실행시 핫키 입력이 씹히는 경우가 더러 있으므로 반드시 F2키로 GTA 내에서 핫키 입력이 잘 되는지 확인 후에
사용하십시오.

  GTA 프로세스 종료 요청 : Esc + F1
  멤버간 신호 송수신 TEST 및 접속멤버 확인 : F2
  프로그램 종료 : Ctrl + Esc 혹은 우측 상단 창 닫기 (서버프로그램은 창 닫기로 종료하십시오.)
